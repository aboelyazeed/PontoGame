### Prerequisites

- Node.js (v18 or higher recommended)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd PontoGame
```

2. Install dependencies:
```bash
npm install
```

### Running the App

- **Start the development server**:
```bash
npm start
```

- **Run on Android**:
```bash
npm run android
```

- **Run on iOS** (macOS only):
```bash
npm run ios
```

- **Run on Web**:
```bash
npm run web
```
